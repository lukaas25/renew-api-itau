
# Renovar credenciais Certificado Dinamico ITAÚ
OBS: Somente para certificado ainda válidos, dentro dos 60 últimos dias válidos

- Script em NodeJS 
- OpenSSL instalado
## Base de Conhecimento

 - [Documentação Itaú](https://devportal.itau.com.br/certificado-dinamico#renovar%20um%20certificado%20ainda%20v%C3%A1lido%20dentro%20do%20prazo%20de%20validade)


## Executar Script

Instale as dependencias

```bash
  npm install 
  ```
Executar o index.js

  ```bash
  node index.js 
  ```
